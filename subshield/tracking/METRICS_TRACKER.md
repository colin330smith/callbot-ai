## IMPORTANT FORMULAS

### Customer Acquisition Cost (CAC)
```
CAC = Total Marketing Spend / New Customers Acquired
```

### Lifetime Value (LTV)
```
LTV = ARPU × Average Customer Lifespan (months)
LTV = ARPU / Monthly Churn Rate
```

### Monthly Churn Rate
```
Churn Rate = Customers Lost / Customers at Start of Month
```

### MRR Growth Rate
```
Growth Rate = (New MRR - Churned MRR) / Previous Month MRR
```

### Conversion Rate
```
Conv Rate = Conversions / Total Visitors × 100
```
